# CAD-бэкенд: CadQuery

Утилита генерирует STL чисто параметрически (скруглённый прямоугольник + выпуклый текст из TTF + цилиндрический вырез). Рассматривались три варианта: OpenSCAD (внешний бинарник + свой DSL), build123d и CadQuery (оба — Python-обёртки над OCCT). Выбран CadQuery: pip-устанавливается без внешних зависимостей (в отличие от OpenSCAD), и у него заметно больше публичных примеров и stackoverflow-ответов, чем у более молодого build123d, — это снижает риск застрять на нестандартных кейсах (импорт глифов из TTF, артефакты булевых операций, параметры тесселяции при экспорте).
